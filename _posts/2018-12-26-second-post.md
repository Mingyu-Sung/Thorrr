# SSAFY Start Camp 챗봇 퀘스트

구미_2_이석원, (결석으로 인한 프로젝트 부재)

# I. 스펙(Specification)
구현된 어플리케이션의 주요 기능

(1) 실시간 검색어 가져오기 기능

* 사용자가 "실시간 검색어 알려줘"라고 채팅창에 입력하면 챗봇이 네이버 실시간 검색어를 가져와서 알려줌.

# II. 회고(Retrospective)

* 집안 일로 인해 수업에 참석하지 못하여 실제로 구현하지 못한 점이 아쉬움.

# III. 보완 계획(Feedback)
* 챗봇 구현을 개인적으로 다시 살펴보면서 실제로 구현을 하고 싶음.
